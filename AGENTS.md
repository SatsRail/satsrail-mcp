# SatsRail MCP Server — AGENTS.md

MCP (Model Context Protocol) server for AI tool integration with SatsRail. Allows AI agents to interact with the SatsRail API through standardized tool calls.

## Tech Stack

- **Language:** TypeScript
- **Runtime:** Node.js
- **Protocol:** MCP (Model Context Protocol)
- **Package:** npm

## Conventions

- Use `npm` (not yarn)
- Tools mirror the SatsRail API resource structure
- See `../api-docs/` for the full OpenAPI spec
- See `../api-docs/guides/ai-agents/` for the AI integration guide
